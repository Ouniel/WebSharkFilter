文件提取技巧

提取压缩包

  • 使用文件魔数过滤特定压缩格式：ZIP(504B0304), RAR(52617221), 7Z(377ABCAF)
  • 结合协议过滤提高准确性：(frame contains "504B0304") and http
  • 对于大文件，使用TCP流重组：tcp contains "504B0304"

重组文件步骤

  1. 应用过滤器定位包含文件的数据包
  2. 右键点击相关数据包 → Follow → TCP Stream
  3. 在TCP流窗口设置正确的方向（客户端/服务器）
  4. 将数据保存为原始二进制文件
  5. 修改文件扩展名为正确的格式（.zip, .rar等）
# 提取HTTP传输的ZIP文件
